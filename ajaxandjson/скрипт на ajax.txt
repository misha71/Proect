Основная цель - научиться работать с ajax в формате json

Слева будут новости, а справа можно будет их редактировать

Необходимые файлы:
Файл index.php (файл отображающий список статей, форму добавления и редактирования статей, а также отправляющий AJAX запросы).
addArticle.php (файл получающий AJAX запрос для добавления статьи со значениями и возвращающий результат добавления).
readArticle.php (файл получающий ID статьи и возвращающий ее данные — поля из БД в формате JSON).
updateArticle.php (файл получающий все поля статьи и обновляющий запись по ID статьи)
